# remarcable-assessment
A Django project to model products, tags, and categories.

Note: db.sqlite3 is intentionally included in this repository as it contains the required sample data (5 categories, 10 tags, 20 products) as specified in the assignment requirements.

## How to run
Clone repo:
``` git clone https://github.com/gkbamrah/remarcable-assessment.git 
cd remarcable-assessment```
